These examples have been copied from the earthcubearchitecture-ecresourcereg/infomodel/examples GitHub repository to be packaged in the first ecrr ontology release

The sitemap.xml file is here to enable testing of JSON-LD harvest agents.

Template files (ECRRTemplate......JSON)
ECRRTemplateSDO.JSON  -- starter template for any registry resource.
ECRRTemplateSDO -Service-v2.JSON --  Template for Service resource description
ECRRTemplateSDO-Software.JSON -- Template for software resource description

the Example instances are named with the resource type they describe, and then a label for the specific resource described.
